# 프로젝트 계획

## 목표
Google AdSense 광고 수익을 목적으로 하는 웹사이트 제작

## 아이디어
주식/코인 트레이딩 봇 수익률 경쟁 플랫폼

- 개인이 제작한 봇을 업로드해서 수익률로 순위를 경쟁
- 일별 / 주별 / 월별 / 연별 기간별 랭킹 제공
- 주식 봇 랭킹 페이지 / 코인 봇 랭킹 페이지 분리

## 페이지 구성
- 로그인 화면
- 주식 봇 랭킹 페이지
- 코인 봇 랭킹 페이지

## 수익화
- Google AdSense (금융 카테고리 — 광고 단가 높음)

## 기술 스택
- Frontend: Vanilla HTML / CSS / JS
- Backend/DB: Firebase (Hosting, Firestore, Auth)

## NOTE (미결정 사항)
- [ ] 봇 업로드 방식 미결정
  - 후보 A: 거래내역 CSV 업로드 (구현 쉬움, 조작 가능성 있음)
  - 후보 B: 거래소 API 키 연동 — 업비트/빗썸 등 (신뢰도 높음, 구현 복잡)
  - 후보 C: 봇 코드 직접 실행 (완전 자동화, 보안/서버 비용 문제)
- [ ] 실제 거래 vs 모의(페이퍼) 트레이딩 여부
- [ ] 수익률 계산 기준 표준화 방식
